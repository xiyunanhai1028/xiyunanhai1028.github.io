'轻松玩转正则'
正则
正则
发布日期:   2017-11-21
阅读次数:  

1. 什么是正则

正则就是一个规则，用来处理字符串的规则

1.1 正则匹配

编写一个规则，验证某个字符串是否符合这个规则，正则匹配使用的是test方法

1.2 正则捕获

编写一个规则，在一个字符串中把符合规则的内容都获取到，正则捕获使用的方法：正则的exec方法，字符串中的split，replace，match等方法都支持正则

两个斜杠中间包含一些内容就是正则，两个斜杠之间包含的全部内容都是元字符

var reg=/^$/;

2. 正则的元字符和修饰符

任何一个正则都是有元字符和修饰符组成

2.1 修饰符

  • g(global)：全局匹配
  • i(ignoreCase)：忽略大小写匹配
  • m(multiline)：多行匹配

2.2 元字符

2.2.1 量词元字符
  • +：上前面的元字符出现一次到多次
  • ?：出现零到一次
  • *：出现零导多次
  • {n}：出现n次
  • {n,}：出现n到多次
  • {n,m}：出现n到m次
2.2.2 特殊意思的元字符
  • \：转义字符（把一个普通字符转变为有特殊意义的字符，或者把一个有意义字符转换为普通的字符）
  • .：除了\n（换行符）以外的任意字符
  • \d：匹配一个0~9之间的数字
  • \D：匹配任意一个非0-9之间的数字（大写字母和小写字母的组合正好互斥）
  • \w：匹配一个0-9或字母或_之间的字符
  • \s：匹配一个任意空白字符
  • \b：匹配一个边界符
  • x|y：匹配x或y中的一个
  • [a-z]：匹配a-z中的任意一个字符
  • [^a-z]：和上面的相反，匹配任意一个非a-z的字符
  • [xyz]：匹配x或y或z中的一个字符
  • [^xyz]：匹配除了xyz以外的任意字符
  • ()：正则的小分组，匹配一个小分组（小分组可以理解为大正则中的一个小正则）
  • ^：以某一个元字符开始
  • $：以某一个元字符结束
  • ?:：只匹配不捕获
  • ?=：正向预查
  • ?!：反向预查

3. 元字符详细解释

3.1 ^ $

包含一个或多个数字

var reg=/\d+/
var str='ab人123cd'
reg.test(str)======>true

str='abc'
reg.test(str)======>false

以一个或多个数字开头

var reg=/^\d+/
var str='12abc'
reg.test(str)=======>true

str='abc12'
reg.test(str)=======>false

只能是1到多个数字

var reg=/^\d+$/
reg.test('100')======>true
reg.test('1a2')======>false
reg.test('1')========>true  ^或者$知识一个修饰或者声明，不会占据字符串的位置

3.2 \

特殊字符不转义的效果

var reg=/^2.3$/
reg.test('2.3')======>true
reg.test('2+3')======>true
点在正则中的意思：匹配除了\n之外的任意字符，而不是单纯的小数点

正确的写法

var reg=/^2\.3$/
reg.test('2.3')======>true
reg.test('2+3')======>false
使用转义字符把点转义为本身小数点的意思

3.3 []

[xyz]，[^xyz]，[a-z]，[^a-z]

数字字母下划线中的任意一个字符

var reg=/\w/

var reg=/^[a-zA-Z0-9_]$/

中括号中出现的元字符，一般都代表本身的含义
var reg=/^[.?+&]+$/
里面的四个元字符都是本身的含义，例如：点就是小数点

描述样式类名的规则（数字，字母，下划线，-)

var reg=/^[\w-]+$/

var reg=/^[a-zA-Z0-9_-]+$/

描述样式类名的规则（数字，字母，下划线，-），并且不能以-开头

var reg=/^\w[\w-]*$/

3.4 ()

正则中的分组，也可以理解为一个大正则中的一个正则（包起来的部分是一个整体）；在正则中我们可以使用小括号改变一些默认的优先级

  • 小分组还有第二个作用：分组引用
  • 小分组的第三个作用：分组捕获

分组引用：

var reg=/^([a-z])([a-z])\2([a-z])$/
\1 或者 \2 ...出现和N个分组一模一样的内容

上面符合条件的：foot,book,week....

验证18~65之间的年龄

/**
*18~19
*20~59
*60~65
*/
var reg=/^((18|19)|([2-5]\d)|(6[0-5]))$/
